## Features

### ▲ PROJECTS
- Most Recent Projects at a glance
- Browse All Projects on your team(s)

### ▲ DEPLOYMENTS
- Actions: Re-deploy, Promote, Rollback, and Cancel
- Distinguish the Current Live Deployment
- View Build Logs, Outputs, Source File Inputs and Functions
- Assigned Domains for Deployment (branch & commit specific)

### ▲ DOMAINS
- Actions: Add and Remove
- Domain Configuration Status (Valid/Invalid)
- Redirect Details
- Tap to copy for DNS details

### ▲ LOGS
- Build Logs in real time
- Function & Request Logs for any Project
- Expand Log Details
- Filters Galore

### ▲ OBSERVABILITY & FIREWALL
- Keep an eye on activity
- Respond to anomalies immediately

### ▲ ENVIRONMENT
- Add or Remove Env Variables
- Edit Environment Variables
